# Active OpenSpec Changes

This folder replaces ExecPlans.

Each active change is both:

- a proposal for what should change;
- the implementation checklist the agent follows after approval.

Create changes as:

```text
openspec/changes/<verb-led-change-id>/
├── proposal.md
├── tasks.md
├── design.md
└── specs/<capability>/spec.md
```

Use `openspec/changes/archive/` for completed changes.
